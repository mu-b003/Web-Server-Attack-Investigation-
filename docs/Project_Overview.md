# Project Overview

## Description

This project demonstrates a complete web server attack investigation performed inside an isolated virtual laboratory.

The objective was to simulate common web attacks against an Apache web server and analyze the generated logs as a Security Operations Center (SOC) analyst.

The investigation focused on identifying attacker behavior, reconstructing the attack timeline, extracting Indicators of Compromise (IOCs), and documenting the incident.

---

## Objectives

- Deploy an Apache web server
- Build a fake web application
- Simulate attacker activity
- Generate Apache logs
- Investigate web server logs
- Identify suspicious activity
- Produce an incident report
